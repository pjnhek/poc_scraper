## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet -> streamable HTTP endpoint | Untrusted HTTP requests (Host/Origin headers, IP headers, JSON bodies) reach the server once Phase 13 deploys; this plan defines the local posture |
| Browser -> localhost server | DNS-rebinding attacks target locally-bound HTTP servers via malicious pages |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-11-09 | Spoofing | DNS rebinding against the HTTP endpoint (CVE-2025-66414/66416 class) | high | mitigate | Explicit TransportSecuritySettings allowlist limited to 127.0.0.1:{port} and localhost:{port} with enable_dns_rebinding_protection=True (Task 1); 421 rejection verified (Task 3 Test 4) |
| T-11-10 | Information Disclosure / Elevation | Accidental network exposure of the local server | medium | mitigate | Default bind 127.0.0.1 (D-09); 0.0.0.0 requires an explicit env override or the Phase 13 Dockerfile, never a local default |
| T-11-11 | Tampering | Header-spoofed bucket evasion through the real HTTP stack | high | mitigate | End-to-end verification that headers resolve through resolve_client_ip's fail-closed rules over the actual transport (Task 3 Tests 2-3); Fly-Client-IP is edge-set in production so clients cannot forge it past Fly |
| T-11-12 | Denial of Service | Session-state accumulation on an unauthenticated endpoint | low | mitigate | stateless_http=True: no long-lived per-session tasks or memory streams (Task 1) |
| T-11-SC | Tampering | npm/pip/cargo installs | low | accept | No new packages this phase (RESEARCH.md Package Legitimacy Audit: none; uvicorn/starlette are existing transitives of the vetted mcp pin) |

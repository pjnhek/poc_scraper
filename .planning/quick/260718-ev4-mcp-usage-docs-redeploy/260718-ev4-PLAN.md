# Quick Task 260718-ev4: Document MCP usage and redeploy

## Problem

Two defects, one causing the other to matter.

**1. The live box is a milestone behind.** `/opt/poc-scraper/repo` on the Oracle host is at `3c5c846` (end of v1.1) while `origin/main` is at `7f960df` (end of v1.2). Verified by probing the live endpoint:

- `tools/list` returns only `get_account_evidence`; `score_account` is absent.
- `get_account_evidence` inputSchema has only `domain`; `news_days` is absent.
- The `research_account` prompt still carries its pre-v1.2 description.
- The landing page still reads "Evidence retrieval only ... The full pipeline (ICP scoring, personas, citation-checked outreach) is BYOK".

`README.md:24` already promises visitors that "deterministic ICP scoring via `score_account` runs on top of it unrationed". That promise is currently false for anyone who connects.

**2. Neither surface documents usage.** Connecting is covered well (four client configs in both README and landing page). Using it is not:

- The landing page's only usage guidance is one sentence in a grey `.note`, and it never mentions the `research_account` prompt, which is the shortest path to value and is already live.
- Neither surface mentions the `icp://rubric` or `icp://eval-report` resources as things a visitor can read.
- Neither surface shows what a response looks like, so a reader never sees the numbered citations that are the project's entire thesis.

## Approach

Deploy first, because an honest sample response has to be captured from a working server rather than invented. The project's core value is that no claim is fabricated; that discipline applies to its own documentation.

## Tasks

### Task 1: Redeploy the Oracle box

- **files:** none (deploy only)
- **action:** Run `make deploy-oracle ORACLE_HOST=170.9.7.144.sslip.io SSH_KEY=~/.ssh/poc_scraper_oracle`. The script pulls `origin/main` into `/opt/poc-scraper/repo`, rebuilds the image, and replaces the container and Caddy config. Swap (2GB) is already present on the 1GB box, so the `uv sync` build step has headroom.
- **verify:** `tools/list` on `https://170.9.7.144.sslip.io/mcp` returns `score_account`; `get_account_evidence` inputSchema contains `news_days`; `prompts/list` shows the six-step `research_account` description.
- **done:** Live surface matches `origin/main`.

### Task 2: Capture a real sample response

- **files:** none (capture only)

- **done:** A real payload is available to quote in the docs.
